# To-Do App - 3

A simple to-do list app made in React that allows users to add and remove tasks.

##  Features
- Add new tasks
- Delete tasks
- Live task list
- useState-based state management

##  Tech Stack
- React.js
- JavaScript
- CSS

## How to Run
1. Clone this repo
2. Run `npm install`
3. Run `npm start`
